# Task Update - Azteca Foods AI Chatbot

## What I'm Working On

I'm building an AI-powered customer support chatbot for Azteca Foods restaurants that helps customers find locations, menu information, and answers to their questions. The chatbot uses Azure AI to provide natural, conversational responses through a modern web widget. I've created a floating chat button with smooth animations that connects to a Python backend server, which handles Azure authentication and processes customer conversations. The chatbot is designed to respond like a human customer service representative, asking clarifying questions when needed and providing helpful information without technical jargon. The solution is complete and ready for deployment.
